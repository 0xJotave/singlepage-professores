# Sistema Acadêmico - Single Page Application (SPA)

Este projeto consiste em uma aplicação de página única desenvolvida para a disciplina de Desenvolvimento Web. O objetivo é listar informações de docentes, suas respectivas disciplinas e fornecer um canal de comunicação direta via formulário de contato.
Funcionalidades

- **Navegação SPA:** Troca de conteúdo dinâmica sem recarregamento de página.
- **Página Inicial:** Apresentação geral do sistema.
- **Perfis de Professores:** Visualização de dados de 3 professores cadastrados.
- **Listagem de Disciplinas:** Exibição detalhada de curso e semestre por disciplina.
- **Formulário de Contato:** Sistema de captura de mensagens com confirmação via modal.

## Tecnologias Utilizadas
- **React.js:** Biblioteca principal para construção da interface.
- **JavaScript:** Lógica de manipulação de estados e filtragem de dados.

## Estrutura do Projeto

A lógica da SPA foi implementada através de renderização condicional baseada no estado da aplicação, garantindo que o cabeçalho permaneça fixo enquanto o conteúdo central é alternado conforme a seleção do usuário no menu.

## Como Executar o Projeto

Clone o repositório para sua máquina local:
```
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

Acesse o diretório do projeto:
```
cd nome-do-repositorio
```

Instale as dependências necessárias:
```
npm install
```

Inicie a aplicação em modo de desenvolvimento:
```
npm start
```

A aplicação estará disponível no endereço http://localhost:3000.
